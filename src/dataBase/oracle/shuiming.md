## 概念

### Oracle11g的database 和client的区别
database 是数据库，你安装到本机上，可以在本机上创建数据库。
client是客户端，你只能通过client连接上别人机器上的数据库。
 
database 包含client的功能
 
 
前者是安装在服务器上的（当然也可以装在你自己的机器上），后者是连接工具的支持文件。
