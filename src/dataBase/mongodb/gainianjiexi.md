| **SQL术语/概念** | **MongoDB术语/概念** | **解释/说明** |
| --- | --- | --- |
| database | database | 数据库 |
| table | collection | 数据库表/集合 |
| row | document | 数据记录行/文档 |
| column | field(字段) | 数据字段/域 |
| index | index | 索引 |
| table joins |  聚合操作($lookup) | 表连接,MongoDB不支持 |
| primary key | id（字段） | 主键,MongoDB自动将_id字段设置为主键 |
