## 软件推荐

### Terminal

一个命令行工具

```sh
# 连接linux
ssh IP -l root
```

美化方案：https://zhuanlan.zhihu.com/p/354603010

### Windows Sandbox

windows沙箱

开启方式：通过控制面板、程序、启用或者关闭功能中勾选windows 沙箱进行开启。

功能：开启后是一个新系统，关闭后里面的东西将释放。

### optimizer
windows优化器，这是一个高级配置实用程序，旨在增强您在 Windows 上的隐私和安全性。强烈建议在全新安装 Windows 后使用此工具，以实现最大的隐私和安全优势。根据您的Windows版本，优化器还可以帮助您应用特定的系统调整。

仓库地址：[https://github.com/hellzerg/optimizer](https://github.com/hellzerg/optimizer)

### FixWin11

FixWin11是一款专为Windows11系统所推出的修改工具。该软件轻巧简便，能够帮助用户快速修复Windows 11各种故障，有效改善用户体验。软件单文件，无需安装（网上有汉化版本）
