## 概述
Python 是一个高层次的结合了解释性、编译性、互动性和面向对象的脚本语言。

**Python 是一种解释型语言：** 这意味着开发过程中没有了编译这个环节。类似于PHP和Perl语言。

**Python 是交互式语言：** 这意味着，您可以在一个Python提示符，直接互动执行写你的程序。

**Python 是面向对象语言:** 这意味着Python支持面向对象的风格或代码封装在对象的编程技术。

**Python 是初学者的语言**：Python 对初级程序员而言，是一种伟大的语言，它支持广泛的应用程序开发，从简单的文字处理到 WWW 浏览器再到游戏。

## 教程

- **前期**：花点时间选一门口碑上佳的入门电子文字教程，坚持下去，不轻易换教程

- **中期**：跟着教程，先学会抄代码，从抄代码中学习代码思维，举一反三，在这个阶段要做好学习笔记，从零开始构建自己的 Python 知识库，方便后面速查。

- **后期**：去某课网上找 Python 的项目实战课堂，一门小几百，质量很高，到这个阶段，要面向工作学编程，从项目中去巩固前面的基础，不断查缺补漏，学习项目开发完整流程。

## 新项目创建

###  安装环境

电脑如果配置了uv，那么跳过该章节，没有搞过的按照安装=>UV的文档去配置

### 创建项目

同样使用uv管理项目的依赖包，其中`pyproject.toml`文件示例如下

```toml
[project]
name = "apiserver"
version = "0.1.0"
description = "Add your description here"
readme = "README.md"
requires-python = ">=3.13"
dependencies = [
    "fastapi==0.124.4",
    "langchain>=1.1.3",
    "langchain-community==0.4.1",
    "langchain-openai>=1.1.3",
    "langchain-postgres>=0.0.16",
    "langgraph>=1.0.5",
    "psycopg2>=2.9.10",
    "psycopg2-binary>=2.9.11",
    "pydantic==2.11.4",
    "python-dotenv==1.1.0",
    "requests==2.32.5",
    "sqlalchemy>=2.0.45",
    "uvicorn==0.34.2",
]

[[tool.uv.index]]
url = "https://mirrors.aliyun.com/pypi/simple/"
default = true
```

### 打包部署

镜像文件示例如下

```dockerfile
# 构建阶段
FROM registry.cn-hangzhou.aliyuncs.com/zrng/python:3.13-slim AS builder

WORKDIR /app

# 设置环境变量
ENV PYTHONUNBUFFERED=1 \
    PYTHONDONTWRITEBYTECODE=1 \
    UV_CACHE_DIR=/tmp/uv-cache

# 配置镜像源并安装构建依赖（合并为单层）
RUN sed -i 's@deb.debian.org@mirrors.ustc.edu.cn@g' /etc/apt/sources.list.d/debian.sources && \
    apt-get update && \
    apt-get install -y --no-install-recommends gcc python3-dev && \
    rm -rf /var/lib/apt/lists/*

# 配置 pip 并安装 uv（合并为单层）
RUN pip config set global.index-url https://mirrors.aliyun.com/pypi/simple/ && \
    pip install --no-cache-dir uv

# 只复制依赖文件以利用缓存
COPY pyproject.toml uv.lock ./

# 创建虚拟环境并安装依赖（使用 lock 文件确保版本一致性）
RUN uv venv .venv --python $(which python3) && \
    . .venv/bin/activate && uv sync --frozen

# 运行阶段
FROM registry.cn-hangzhou.aliyuncs.com/zrng/python:3.13-slim

WORKDIR /app

# 设置环境变量
ENV PYTHONUNBUFFERED=1 \
    PYTHONDONTWRITEBYTECODE=1 \
    PATH="/app/.venv/bin:$PATH" \
    LOG_LEVEL=INFO \
    HOST=0.0.0.0 \
    PORT=5000

# 从构建阶段复制虚拟环境
COPY --from=builder /app/.venv /app/.venv

# 复制应用代码
COPY . .

# 暴露端口
EXPOSE 5000

# 直接启动 uvicorn
CMD uvicorn main:main_app --host 0.0.0.0 --port 5000 --workers 4
```
