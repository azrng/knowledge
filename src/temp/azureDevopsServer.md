## 概述

Azure DevOps Server 是微软提供的一个应用软件生命周期管理产品。它是一个强大的软件开发管理平台，支持软件开发全流程，包括需求管理、项目管理、源代码版本管理、自动化构建与测试、以及制品管理等功能。

文档：[https://learn.microsoft.com/zh-cn/azure/devops/server/?view=azure-devops](https://learn.microsoft.com/zh-cn/azure/devops/server/?view=azure-devops)

### Azure DevOps Services和Azure DevOps Server区别

云 **产品/服务** Azure DevOps Services 提供可缩放、可靠且全局可用的托管服务。

本地 **产品/服务** Azure DevOps Server 基于 SQL Server 后端构建。 客户通常需要将数据保留在其网络内时，通常会选择本地版本。 或者，当他们想要访问与 Azure DevOps Server 数据和工具集成的 SQL Server Reporting Services 时。

## 安装

我们可以从这里进行[下载](https://learn.microsoft.com/en-us/azure/devops/server/download/azuredevopsserver?view=azure-devops#download-the-latest-release)，我只是本地测试所以直接选择Azure DevOps Server Express 2022.2的exe文件即可

> **Azure DevOps Server Express** 是免费的，易于在客户端和服务器作系统上进行设置，并支持与 Azure DevOps Server 相同的所有功能。唯一的区别是，它受许可协议限制为 5 个或更少的活动用户。
