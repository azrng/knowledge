## 概述

Kiota 是一个命令行工具，用于生成 API 客户端以调用您感兴趣的任何 OpenAPI 描述的 API。目标是消除对需要调用的每个 API 依赖不同 API SDK 的需要。Kiota API 客户端提供强类型体验，具有您期望从高质量 API SDK 获得的所有功能，但无需为每个 HTTP API 学习新库。

文档：https://learn.microsoft.com/zh-cn/openapi/kiota/overview
