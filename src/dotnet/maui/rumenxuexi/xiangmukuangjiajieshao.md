## 概述
- MauiProgram.cs文件包含用于创建和配置应用程序对象**的**代码。
- 提供 UI 资源并为应用程序创建初始窗口的**App.xaml**和App.xaml.cs**文件。**
- AppShell.xaml和**AppShell.xaml.cs**文件指定应用程序的初始页面并处理导航路由页面的注册**。**
- MainPage.xaml和**MainPage.xaml.cs**文件定义默认显示在初始窗口中的页面的布局和 UI 逻辑**。**
