## 介绍
Razor是一个用于将基于.NET的代码嵌入到网页中的标记语法。 Razor语法由 Razor 标记、C#代码 和 HTML 组成。

## 常用组件

- RazorEngine
- RazorEngine.NetCore
- Microsoft.AspNetCore.Mvc.Razor
- RazorLight
