## 概述

Avalonia是一个强大的框架，使开发人员能够使用.NET创建跨平台应用程序。它使用自己的渲染引擎绘制UI控件，确保在Windows、macOS、Linux、Android、iOS和WebAssembly等不同平台上具有一致的外观和行为。这意味着开发人员可以共享他们的UI代码，并在不同的目标平台上保持统一的外观和感觉。

官网地址：[https://www.avaloniaui.net/How-It-Works](https://www.avaloniaui.net/How-It-Works) 

中文官网：[https://docs.avaloniaui.net/zh-Hans/docs/next/welcome](https://docs.avaloniaui.net/zh-Hans/docs/next/welcome)  

在线编辑测试：[https://play.avaloniaui.net/ ](https://play.avaloniaui.net/)

## 快速上手

### 安装模板

运行命令安装模板，详细步骤看[此处](https://docs.avaloniaui.net/zh-Hans/docs/get-started/install)

```csharp
dotnet new install Avalonia.Templates
```

### 安装可视化界面

比如Vs中可以安装：[Avalonia for Visual Studio 2022](https://marketplace.visualstudio.com/items?itemName=AvaloniaTeam.AvaloniaVS) 以及XAML格式化工具[XAML Styler for Visual Studio 2022](https://marketplace.visualstudio.com/items?itemName=TeamXavalon.XAMLStyler2022)，或者在Rider中安装[AvaloniaRider](https://plugins.jetbrains.com/plugin/14839-avaloniarider)

### 热重载

使用[HotAvalonia](https://github.com/Kira-NT/HotAvalonia)进行热重载
