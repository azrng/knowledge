## 概述
工作服务：工作服务是一种基于任务调度的服务执行机制。与Windows服务不同，工作服务通常是由应用程序或框架提供的一种机制，用于定期或按需执行特定的任务。工作服务不会在系统启动时自动启动，而是由应用程序根据需要手动触发或按计划执行。
