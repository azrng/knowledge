资产划分三份，买入不同的内容

1、保持灵活和流动性，收益能超过2倍余额宝就行，所以买7免、不限购、支持转换的债券

2、要有合适收益保障，愿意放弃流动性。可以接受长期。所以买长期免赎期、甚至有申购费的债券，A类也可以考虑

3、追求高收益，愿意承担风险，尝试参与一些波段操作。比如买些股票基金或者纳指和黄精，做一些多元化操作。

## 债券型基金

### 时间段

法师经验之谈：每年3、6、9、12月都是调整月

### 如何选择债基

1、选择发行时间久的。 
2、看债基规模不是很小时，看债基是否限购。如果限购，正常来说基金公司是比较看重这个产品的，是想把其做好来吸引客户的 
3、看债基各个时间段的同类排名。排名有靠后、一般、普通、优秀之分，选择收益率同类排名优秀的时间段多的。 
4、看债基最近的收益率：尽量选择近1周或近1月收益率同类排名是优秀的 
5、选择规模适中的债基：规模太大，会摊薄超额收益。规模太小，可能会被清盘  

### 长中短债区别

债券型基金有长期纯债和短期纯债之分，短期纯债还细分为中短债、短债和超短债。

长债基金的产品名字通常叫做某某纯债或者某某信用债；中短债、短债和超短债的产品名称通常直接叫做某某中短债、某某短债和某某超短债。



长债基金投资的债券一般在一年到五年左右，中短债基金投资的债券一般在1到3年内，短债投资的债券一般在一年内，超短债券投资的债券一般在270内

### 信用债和利率债

简单理解：利率债无信用风险，信用债有信用风险。

| 区别     | 利率债                                   | 信用债                                       |
| -------- | ---------------------------------------- | -------------------------------------------- |
| 发行主体 | 国家或信用等级和国家相同的结构           | 企业、公司                                   |
| 利率风险 | 有                                       | 有                                           |
| 信用风险 | 无                                       | 有                                           |
| 种类     | 国债、政策性金融债、央行票据、地方政府债 | 企业债、企业短期融资债、中期票据、同业存单等 |

### 信用债分类

信用债主要有银行债、城投债、产业债之分。根据时间又能分为短债、中票、长债。

## 黄金

### 黄金ETF与上海金ETF区别

投资黄金的主流方式除了实物黄金(金条)、纸黄金，还有就是黄金ETF。

上海金ETF主要投资“上海金集中定价合约”和“黄金9999现货合约”，投资范围更广

黄金ETF主要投资黄金9999现货合约。两者的投资标确实不太一样。

但是投资标不一样不是造成价格差异的主要原因，因为这两种黄金斗士以纯度99.99%的金锭计价，而且用的都是本国货币，规避了汇兑影响。

价格有差异的主要元婴是：上海金的“集中定价合约”是按照15:00来结算的，而黄金ETF的现货合约，是以15:30结算。这半个小时，有时候会对当日收益产生比较大的影响。如果长期持有的话，二者的收益差异非常小。

## 买入卖出

### 如何区分小波动大调整

若跌幅在0.01%到0.03%，那就属于小幅波动，我会继续持有不动，静待回调结束。

若跌幅在0.04%到0.06%，连续两天在此跌幅区间，并且第三天故事也在这个区间，那么我就会把没有手续费的转到货币基金或卖出到余额宝避险，待稳定后再回来。

### 成本计算

补仓后成本计算公式

```
补仓后成本 = (第一次买入基金的成本价*第一次买入的数量+补仓基金的成本价*补仓数量)/(第一次买入数量+补仓数量)
```

比如，投资者在基金净值为1.5元的时候，买入基金1000份，买入之后，基金出现下跌的情况，当基金净值下跌1.0元的时候，在补仓2000股，则补仓之后，投资者的持仓成本=(1.5\*1000+1.0\*2000)/(1000+2000)=1.167元，通过补仓之后，其成本下降了0.333元。

## 上车操作

:::tip

摘录自：发牌手杰克

:::

打个比方，目前你准备了1万块买基金，看中了某只基金，准备要操作了，下面就是你即将遇到的问题：

1、什么时候买？

任何时候买都是最佳时候，因为涨高了，你不敢动，跌得低了，又怕没抄到底，所以金字塔加仓法，任何时候都可以介入购买。

2、怎么买？

准备了1万，今天直接购入50元，第一天跌了，幅度超过1%，那就补仓100元，第二天如果再跌，那就补仓200元，第三天还跌呢？那就是直接加仓400元，以此类推，你的成本会被大大摊低，等涨起来的时候就是到底吃肉的时候了。

3、买了50之后一路涨怎么办？

这个问题相信大家有可能会碰到，这样子更好，在第一天买入50的时候，我们就开始设置每天的定投，如每天定投50元，一个月的交易日下来大概支出1000左右，一路涨，就一路吃定投的肉，永远都不会踏空。

### 分步建仓法

把手上的闲钱，分成三大份，其中一份再分割成60份（一个月20个交易日左右），开始设置定投，这60份的定投时长为3个月。
打个比方，手上现在有1万，我拿4000出来，切割成60份，大概就是70元，我直接看上一个基金，开始定投70元，期间不进行操作，任其涨跌。
优点：定投可以规避掉人为操作引发的失误，用规则限制住人性，涨的时候可以一直吃肉,跌的时候可以分摊风险。
缺点：分步建仓法属于中庸平稳，短期之内不会有太明显的波动，见效不算快，但是很稳。求稳是我们新手投资者必须做的。

### 金字塔加仓法

三个月过去了，定投停止。手上还有2份各3000元的闲钱。此刻，只要基金每下跌1%或者2%，就开始补仓70元，如果再跌2%，补仓140元，再再跌，补仓280元，以此类推，手动补仓一次不超过上限500元。遇到行情好的时候，一次性补1000元。（规则是死的，人是活的，策略是指导，正常来说不会把子弹打空)
