## 概述
Blazor Spark 是一个开源 Web 框架，旨在通过其富有表现力和直观的语法来加速 Web 应用程序开发。它提供了坚实的基础和一组强大的功能，使开发人员能够轻松创建卓越的 Web 应用程序。
Blazor Spark 采用 Blazor Server 基于组件的架构并扩展了其功能，使开发人员能够快速构建功能丰富的 Web 应用程序。
它还提供无缝且高效的开发体验，包括自定义 CLI 命令以提高您的效率。
文档地址：[https://blazorspark.com](https://blazorspark.com)
