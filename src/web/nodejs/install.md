## 版本管理Nvm

```shell
# 安装 nvm-windows，下载exe文件
https://github.com/coreybutler/nvm-windows/releases

# 查看当前版本
num list

# 卸载当前 Node.js
nvm uninstall 22.15.0

# 安装新版本
nvm install 22.15.0
nvm use 22.15.0
```
