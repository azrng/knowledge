## 商城

### eShop

微软的开源项目 eShop 和 eShopOnContainers 都与电子商务相关，它们之间存在一定的联系：

eShop 是一个简化的电子商务应用示例，旨在展示微软的 .NET 技术栈和 Azure 云平台的最佳实践。它提供了一个基本的在线商店应用的实现，包括产品目录、购物车、订单处理等功能。

eShopOnContainers 是基于微服务架构的电子商务参考应用，同样使用了 .NET 技术栈和 Azure 云平台。eShopOnContainers 将电子商务应用拆分为多个微服务，每个微服务都独立运行，并通过容器化技术来管理。它还包括了诸如身份认证、支付服务、产品目录等微服务的实现。

因此，可以说 eShopOnContainers 是在 eShop 的基础上演化而来，它是对微服务架构和容器化技术的一个示范，并提供了更加复杂和完整的电子商务解决方案。

eshop地址：https://github.com/dotnet/eShop

eShopOnContainers地址：[https://github.com/dotnet-architecture/eShopOnContainers](https://github.com/dotnet-architecture/eShopOnContainers)

eShopOnContainers教程：https://www.cnblogs.com/sheng-jie/category/1319147.html

#### NSmartProxy
NSmartProxy 是一款强大开源的内网穿透工具，采用.NET CORE的全异步模式打造。

1.  跨平台，客户端和服务端均可运行在MacOS，Linux，Windows系统上； 
2.  使用方便，配置简单； 
3.  多端映射，只需安装一个NSmartProxy客户端可映射整个局域网内的多种服务；

### NetEngine 

NetEngine 是一个基于 .Net 搭建的快速开发框架，项目集成了很多常用的模块，开箱即用，这个项目始于2019年，从 .NET Core 2.2 时代一路迭代过来。



项目主要整合了一些常用的技术点，如下

- API 的授权认证采用 JWT 认证方式（JWT自动续期逻辑实现，到期前15分钟会签发新的Token）
- 全局异常记录实现
- Redis分布式缓存实现
- Redis分布式锁实现，支持并发所，信号量锁
- 微信和支付宝支付模块实现
- 微信小程序开发中常用的接口实现，如获取手机号，获取openid 等等
- 微信APP登录获取 token 实现
- 手机号和短信验证获取 token 实现
- 文件存储服务对接了 阿里云OSS和腾讯COS
- 短信服务对接了腾讯云和阿里云短信服务
- 雪花ID的自定义实现，支持 139年 的有效期
- 日志服务扩展，实现了数据库和本地文件两种记录模式
- 采用 Blazor 实现了一个基础的CMS管理后台，拥有完整的角色和用户权限控制

仓库地址：https://github.com/dashiell-zhang/NetEngine

### Navidrome音乐播放器

Navidrome 是一开源的音乐服务器，可以用来自建云端音乐播放器，让你在任何地方通过浏览器或者手机来收听自己的音乐，兼容 Subsonic、Aironic 播放器。

官网：[https://www.navidrome.org/](https://www.navidrome.org/)

docker-compose安装脚本，配置中使用 volumes 指定了配置文件夹 data，歌曲存放文件夹 folder.

```yaml
version: "3"
services:
  navidrome:
    image: deluan/navidrome:latest
    user: 1000:1000 ## should be owner of volumes
    ports:
      - "4533:4533"
    restart: unless-stopped
    environment:
      ## Optional: put your config options customization here. Examples:
      ND_SCANSCHEDULE: 1h
      ND_LOGLEVEL: info
      ND_SESSIONTIMEOUT: 24h
      ND_BASEURL: ""
    volumes:
      - "~/app/music/data:/data"
      - "~/app/music/folder:/music:ro"
```
