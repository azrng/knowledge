## 概述
RediSearch 是一个分布式全文搜索和聚合引擎，构建为 Redis 之上的模块。它使用户能够以极快的方式在其 Redis 数据集上执行复杂的搜索查询。 RediSearch 的独特架构是用 C 语言编写的，并从头开始构建在优化的数据结构上，使其成为市场上其他搜索引擎的真正替代品。它作为一个独立的搜索引擎非常适合用于索引和检索可搜索数据。
