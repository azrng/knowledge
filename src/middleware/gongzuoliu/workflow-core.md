## 概述

Workflow-Core是一个基于.NET Standard的轻量级工作流引擎，目前有超过3100+个star。它提供了FluentAPI、多任务、持久化以及并行处理的功能，适合于小型工作流、责任链的需求开发。
由于Workflow-Core支持工作流长期运行，因此Workflow-Core支持以下多种数据源格式的持久化，可以通过安装不同的Provider包来实现对应的持久化：

- （默认提供，用于测试和开发）内存
- MongoDB
- MS SQL Server
- MySql
- Sqlite
- Redis
- PostgreSQL
